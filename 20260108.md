# How to set the cursor for CachyOs

1. Install the cursor icons (in my case "bibata"):
`yay -S bibata-cursor-theme-bin`
2. Set the theme in ~/.profile and ~/.icons/default:

In ~/.profile, append:
```
export XCURSOR_THEME=Bibata-Modern-Classic
export XCURSOR_SIZE=20
```

In ~/.icons/default/index.theme, append:
```
[Icon Theme]
Name=Default
Comment=Default Cursor Theme
Inherits=Bibata-Modern-Classic
```
In ~/.config/environment.d/10-cursor.conf, append:
```
XCURSOR_THEME=Bibata-Modern-Classic
XCURSOR_SIZE=20
```

3. Log current user out and log back in with:
`loginctl terminate-user $USER`
